Sentiment Predictor

An API that predict sentiments of textual data in English, Urdu or Roman Urdu

Click Here To Predict Sentiment

Access the source code or PyPI package

A multilingual sentiment predictor which classifies English, Urdu or Roman Urdu text as Negative or Positive.
It uses sentiment analysis algorithms of machine learning to classify negative and positive texts. We used 5
different algorithms and then calculated the sum of predictions to vote if a text is most likely to be negative
or positive in sentiment.

The source code for python package is publically available on GitHub.
The pip installable python package is available on PyPI.
